# 🎬 Netflix Content Analytics Dashboard

## 📌 Project Overview

This project focuses on analyzing Netflix’s content dataset to uncover insights related to content type, genre distribution, release trends, and country-wise contributions.
An interactive dashboard was built using **Power BI** to visualize key patterns and support data-driven decision-making.

---

## 🎯 Objectives

* Analyze the distribution of Movies vs TV Shows
* Identify popular genres and content trends
* Study yearly growth of Netflix content
* Understand country-wise content production
* Build an interactive dashboard for business insights

---

## 🛠️ Tools & Technologies

* **Power BI** (Dashboard & Visualization)
* **Power Query** (Data Cleaning & Transformation)
* **DAX** (Calculated Measures & KPIs)
* **Excel** (Initial Data Handling)

---

## 📊 Dataset Information

* Dataset contains **6000+ Netflix titles**
* Includes details like:

  * Title
  * Type (Movie/TV Show)
  * Genre
  * Release Year
  * Country
  * Duration

---

## 🔍 Key Steps Performed

### 1. Data Cleaning & Preprocessing

* Handled missing values and inconsistent entries
* Standardized columns for analysis
* Removed duplicates and corrected data formats

### 2. Data Transformation

* Used Power Query for shaping and structuring data
* Created meaningful categories for better analysis

### 3. Data Analysis

* Performed **Exploratory Data Analysis (EDA)**
* Identified trends in:

  * Content type distribution
  * Genre popularity
  * Year-wise content growth

### 4. Dashboard Development

* Built an interactive Power BI dashboard with:

  * KPIs
  * Bar charts
  * Line charts
  * Slicers and filters

---

## 📈 Key Insights

* Movies dominate the platform compared to TV Shows
* Significant growth in content production after 2015
* Certain genres like Drama and Comedy are most frequent
* USA contributes the highest number of titles
* Content production has increased rapidly in recent years

---

## 🖥️ Dashboard Features

* Interactive filters for year, genre, and country
* Drill-down capabilities for deeper insights
* User-friendly layout for easy navigation
